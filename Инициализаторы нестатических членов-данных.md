# Инициализаторы нестатических членов-данных C++11
Позволяет инициализировать нестатические члены-данных там, где они объявлены.
```c++
// Инициализация по умолчанию до C++11 
class Human { 
		Human() : age(0) {} 
	private: 
		unsigned age; 
}; 

// Инициализация по умолчанию в C++11 
class Human { 
	private: 
		unsigned age{0}; 
};
```
