# std::array C++11
`std::array` – это контейнер, построенный поверх C-подобных массивов. Поддерживает общие контейнерные операции, такие как сортировка.
```c++
std::array<int, 3> a = {2, 1, 3}; 
std::sort(a.begin(), a.end()); // a == { 1, 2, 3 } 

for (int& x : a) 
	x *= 2; // a == { 2, 4, 6 }
```
