# Функции по умолчанию C++11
Обновления C++11 вводят более элегантный и эффективный способ обеспечить реализацию функции по умолчанию, например конструктора.
```c++
struct A {
  A() = default;
  A(int x) : x(x) {}
  int x{ 1 };
};

A a{}; // a.x == 1
A a2{ 123 }; // a.x == 123
```

С наследованием:
```c++
struct B {
  B() : x(1);
  int x;
};

struct C : B {
  // Calls B::B
  C() = default;
};

C c{}; // c.x == 1
```
